name: new-tpl
description: Design a new product template and convert to ZKEACMS template format. Use this skill when the user asks to design a new product template.
---

Step 1: 设计新模板

使用 `frontend-design` SKILL 设计一个新的产品模板。
如果用户没有指定产品类别，就由你来选择一个合适的类别。

Step 2: 转换为ZKEACMS模板格式

使用 `convert-to-cms-tpl` SKILL 将设计好的模板转换为ZKEACMS模板格式。
